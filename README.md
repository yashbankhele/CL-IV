# 🧠 Deep Learning Practical Experiments

This repository contains a set of **Deep Learning practical experiments** performed as part of the academic curriculum. These experiments cover core concepts of neural networks, including CNNs, RNNs, and GANs, with hands-on implementation.

---

## 📑 Table of Contents

| Sr. No | Experiment Title |
|--------|-----------------|
| 1 | House Price Prediction Using Linear Regression |
| 2 | Multiclass Classification using CNN |
| 3 | Time Series Prediction using RNN (LSTM/GRU) |
| 4 | Image Classification using CNN with Hyperparameter Optimization |
| 5 | DCGAN for Image Generation |

---

## 🎯 Objective

- To understand fundamental and advanced concepts of **Deep Learning**
- To implement different neural network architectures
- To gain hands-on experience with real-world datasets
- To evaluate model performance using appropriate metrics

---

## 🧪 List of Experiments

### 🔹 Experiment 1: House Price Prediction
- Algorithm: Linear Regression  
- Type: Supervised Learning  
- Goal: Predict house prices based on input features  

---

### 🔹 Experiment 2: Multiclass Image Classification
- Model: CNN (Convolutional Neural Network)  
- Goal: Classify images into multiple categories  

---

### 🔹 Experiment 3: Time Series Prediction
- Model: RNN, LSTM, GRU  
- Goal: Predict sequential data like stock prices  

---

### 🔹 Experiment 4: CNN with Hyperparameter Optimization
- Focus: Improve model accuracy using tuning techniques  
- Techniques: Learning rate, batch size, dropout  

---

### 🔹 Experiment 5: DCGAN (Image Generation)
- Model: Generative Adversarial Network  
- Goal: Generate realistic images using deep learning
